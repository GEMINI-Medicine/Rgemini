# Plot GEMINI color palettes

Plots all available GEMINI color palettes.

## Usage

``` r
plot_color_palettes(plot_palettes = "all")
```

## Arguments

- plot_palettes:

  (`character` or `numeric`)  
  Which palettes to plot. By default, plots all palettes.
