# Coverage Message

Message to display to inform the user that the function being used does
not check clinical data coverage.

## Usage

``` r
coverage_message(what, addtl = NULL)
```

## Arguments

- what:

  (`character`)  
  Which clinical table is used.

- addtl:

  (`character`)  
  An additional, specific message to append to the generic message.

## Value

Prints a message to the console.
